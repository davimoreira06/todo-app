# Todo app

## Sobre o projeto

https://todo-app-six-eosin-34.vercel.app/

Lista de tarefas onde os usuários podem:
- Criar novas tarefas.
- Marcá-las como concluídas.
- Deletar as tarefas manualmente.
- Deletar todas as tarefas concluídas.
